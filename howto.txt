1. install git

2. dal terminale andare sulla path del progetto

3. dare il comando: git init 

-------------------il comando git init crea una nuova repository. Si da una unica volta questo comando------------

4. dare il comando: git add .

--------------------aggiuge i file alla repository--------------

5. dare il comando: git commit -m "quello che vuoi"

-------serve per far capire quali sono gli aggiornamenti fatti-------------

6. Andiamo su github e creiamo o prendiamo una repository. Una volta creata o entrati in una esistente dobbiamo copiare l'url che ci viene dato. P.S. lascio una immagine esempio dell'url di cui parlo. Così lo trovate quando create una nuova repo sennò dovete andare u CODE(tasto verde) e li dentro trovate il link che è lo stesso che serve per fare un clone.

7. dare il comando: git remote add origin "url"
